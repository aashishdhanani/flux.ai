# flux.ai

## Setup

1. Create a `.env` file in the root directory with the following variables:

   ```plaintext
   PORT=
   MONGODB_URI=
   URL=```

2. Install dependencies

    `npm install`
    `cd client`
    `npm install`

3. Start 
    `mongod --dbpath data/db`
    `npm start`
    `cd client`
    `npm start`
